# Exercise 1: Prompt Builder

## Overview

In this exercise you'll learn to **"ground"** Einstein in your account records so the
assistant speaks only from facts, not hallucinations.

You will build two prompts:

- **1.1 — Smart Account Research:** a Field Generation prompt that turns your research
  notes — plus the web — into a structured **Smart Account Profile**.
- **1.2 — Investor Outreach Email:** a Sales Email prompt that turns that profile into a
  personalized outreach email to your contact.

::: tip Prerequisites
Complete **Exercise 0: Pre-Setup Checklist** first — you need the Proxima Fusion Account
and Lukas Vogel Contact, Einstein enabled, and the Einstein permissions assigned.
:::

::: info Web & research enrichment
The profile is enriched from the **research notes in the record** and from the **web** via
a web search action (configured together during the session). Deeper data enrichment can
be layered on later via Data Cloud or an external service.
:::

Continue to **1.1 Smart Account Research** below.
